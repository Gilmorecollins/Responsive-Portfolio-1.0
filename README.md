# Personal Portfolio Website

This is a personal portfolio website built with HTML, CSS, JavaScript, and Bootstrap. It is a simple and responsive website that showcases my skills and experience.

## Getting Started

To get started, clone the repository and install the dependencies.

```
git clone https://github.com/gilmorecollins/personal-portfolio-website.git
cd personal-portfolio-website
npm install
```

## Running the Website

To run the website, start the development server.

```
npm start
```

The website will be available at http://localhost:3000.

## Code Structure

The code is structured as follows:

* `index.html`: The main HTML file of the website.
* `style.css`: The main CSS file of the website.
* `script.js`: The main JavaScript file of the website.
* `images`: The folder containing the images used on the website.

## HTML

The HTML file is responsible for the structure of the website. It contains the `<head>` and `<body>` sections. The `<head>` section contains the title of the website and the link to the CSS file. The `<body>` section contains the content of the website, such as the navigation bar, the home section, the about section, the services section, the skills section, the teams section, and the contact section.

## CSS

The CSS file is responsible for the styling of the website. It contains the styles for the different elements of the website, such as the navigation bar, the home section, the about section, the services section, the skills section, the teams section, and the contact section.

## JavaScript

The JavaScript file is responsible for the interactivity of the website. It contains the code for the navigation bar, the home section, the about section, the services section, the skills section, the teams section, and the contact section.

## Images

The images folder contains the images used on the website. The images are used in the home section, the about section, the services section, the skills section, the teams section, and the contact section.

## Conclusion

This is a simple and responsive personal portfolio website built with HTML, CSS, JavaScript, and Bootstrap. It is a great way to showcase your skills and experience.
